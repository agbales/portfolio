American Lynching

This project combines the Library of Congress newspsper API with landmark lynching datasets from the past 100 years. Users can see articles and opinion pieces form their own state or explore documented lynching cases that lead them directly to primary sources.

It might surprise you to hear how incomplete and disconnected many lynching data sets are. Researchers continue to piece together statisitics, and this project was a way to further complete that picture with a vast newspaper resource.

I submitted this project to the National Endowment for the Humanities challenge for the Library of Congress database Chronicling America API. Before embarking on this project, I'd been working with Cincinnati Open Data for the past few months and found that the API's offerings of lynching narratives from the 1800s into the early 1900s were both haunting and illuminating to modern day statistics.

  • Chronicling America API (JSON)
  • University of North Carolina - Wilmington, Project HAL (CSV)
  • Hand-coded Tuskegee University Data (CSV)

Cincinnati Police Data

This project uses Cincinnati Open Data to chart citizen complaints against the police department. It turns dense documentation into simple and interactive maps. Users can see gender, racial, and complaint types in their area using a custom SVG map I drew in Illustrator.

In the next stage of development I plan to allow district comparisons. Next, I'll make increasingly specific neighborhood maps that allow users to work with even more localized data.

  • Cincinnati Open Data — SODA API

Roadkill

I'd wanted to work with Google Maps for a while. After I found this strange and rarely viewed data about roadkill pickup in Cincinnati, I jumped right in.

The app charts all entries by their latitude and longitude. When the user clicks on the various icons they reveal text from the report. You'll find it varies from matter of fact to oddly personal.

This was also a chance for me to work with Adobe Illustrator. Each animal icon is custom, made from both tracings and scratch.

University of Cincinnati
April, 2016 - Present

The University of Cincinnati has a stunning collection of recordings that for years haven't been particularly accessible. Each year since the 1950s, nationally renowed writers have visited campus to give reaadings. In fact, the poet Robert Frost was the first to visit campus.

I began the work of making these recordings more accessible. The files lived in a relatively hard to navigate collection in the university's digital archive. Instead, we wanted a new website that would be easily maintainable by student assistants.

For this, I customized a Wordpress page to appeal to both researchers and community members. I then wrote JavaScript and PHP scripts to transfer the bulk of links into their new pages.

  • Managed project from concept to implementation
  • Wrote JavaScript & PHP code to connect database to Wordpress posts
  • Coordinated with UC's Digital Content Stratgist
  • Took photos & video to supplement content
